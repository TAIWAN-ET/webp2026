# Dust2 media placement

Place Dust2 tutorial assets in this folder structure:

- `public/media/dust2/videos/`
- `public/media/dust2/posters/`

Naming convention:

- video: `slug.mp4`
- poster: `slug.jpg` or `slug.webp`

Example:

- video: `public/media/dust2/videos/a-short-to-a-long.mp4`
- poster: `public/media/dust2/posters/a-short-to-a-long.jpg`

The React UI points to paths such as:

- `/media/dust2/videos/a-short-to-a-long.mp4`
- `/media/dust2/posters/a-short-to-a-long.jpg`
